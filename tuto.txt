📑 Tutorial 📑
———————————————-——-
- Use el comando /host para especificar el moodle donde está alojada su cuenta  (ejemplo https://moodle.uclv.edu.cu/).

 - Use el comando /account seguido myusuario,contraseña para configurar su cuenta.

- Use el comando /zips para cambiar el tamaño de  los archivos.

- Use el comando /archivos para examinar los archivos que hay en la nube.

-Presiona /archivos para extraer los enlaces de un archivo.

-Presiona /txt para extraer los txt de un archivo.

- Presiona /borrar para borrar archivos subidos en la nube.

- Presiona el comando /cancel para cancelar una descarga.
